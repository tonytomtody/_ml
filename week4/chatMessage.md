## 📚 建立的文件

### 1. **nn0_example.py** - 獨立可執行範例
4 個完整的訓練任務：
- ✓ **XOR 問題**：展示神經網路如何解決非線性分類問題
- ✓ **多類分類**：3 類高斯分佈資料的分類（達 100% 準確度）
- ✓ **函數擬合**：訓練網路擬合 sin(x) 函數
- ✓ **梯度檢查**：驗證反向傳播實現的正確性（誤差 < 1e-9）

### 2. **nn0_interactive_tutorial.ipynb** - 互動式 Jupyter Notebook
6 大部分，可逐步執行學習：
- 📌 Value 類的自動微分演示
- 📌 MLP 模型實現
- 📌 Adam 優化器訓練迴圈
- 📌 2 個分類任務的準確度評估
- 📌 損失曲線和梯度分佈可視化
- 📌 不同學習率的影響對比

## 🎯 主要功能

| 功能 | 説明 |
|------|------|
| **自動微分** | Value 類支援完整的符號梯度計算 |
| **優化算法** | 實現帶學習率衰減的 Adam 優化器 |
| **激活函數** | ReLU (隱藏層) + Softmax (多類) |
| **損失函數** | MSE (回歸) 和交叉熵 (分類) |
| **驗證方法** | 梯度檢查、準確度評估、損失監控 |
